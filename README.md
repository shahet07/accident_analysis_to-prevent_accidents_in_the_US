# accident_analysis_to-prevent_accidents_in_the_US

## US Traffic Accidents Data Warehouse (Team 4)

This project contains a data warehousing and OLAP analysis workflow based on the **US Accidents** dataset.

## Project Files
- `OLAP.ipynb` - OLAP analysis notebook
- `US_Accidents_Final_Presentation.pptx` - Final presentation
- `US_Traffic_Accidents_Data_Warehouse_Report.pdf` - Project report

## Dataset
The source dataset is from Kaggle:
- [US Accidents Dataset (Kaggle)](https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents?resource=download)

Note:
- Large raw datasets are not committed to this repository.
- Download the dataset from Kaggle and place it locally before running notebook steps that require data.

## How to Run
1. Open `OLAP.ipynb` in Jupyter Notebook/JupyterLab.
2. Update dataset file paths in the notebook if needed.
3. Run cells in order.
